## 🚀 Projects

### 📱 Basketball Scoring App — *Android / Kotlin*
> A real-time sports scoreboard for Android with multi-team tracking and live score updates.
- Built with **Kotlin** in Android Studio
- Real-time score updates with reset functionality
- Practical introduction to Android app lifecycle and UI events

---

### 🗄️ Student Management System — *MySQL / Database Design*
> A multi-schema relational database system for student data management.
- Designed with **MySQL Workbench** across multiple linked schemas
- Modules covering student records, accounting, and health/status data
- Role-based data organization and modular architecture thinking
- Explored **SHA1 hashing** for credential security in linked authentication layer

---

### 🖥️ Desktop GUI Application — *C++ / Qt*
> A desktop software project exploring Qt-based GUI structure and secure login design.
- Built with **Qt Creator** using C++
- UI-driven desktop application architecture
- Authentication flow with **SHA1 password hashing** for credential protection
- File-based project structure and modular component organization
